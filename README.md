Personal Expense Tracker
A simple Python project to track daily expenses, analyze spending by category, and visualize your expenses with charts. This project uses a MySQL database to store your expenses and Python functions to manage, analyze, and display your data.

---

Highlights
- Add daily expenses with amount, category, and date  
- View all expenses in a table  
- Calculate total spending  
- Analyze spending by category  
- Visualize spending with pie and bar charts  
- Interactive menu-driven interface  

---

Technologies used
- Python  
- MySQL  
- PyMySQL  
- Matplotlib  
- Jupyter Notebook
